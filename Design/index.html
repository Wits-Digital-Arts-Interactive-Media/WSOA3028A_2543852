The designs shown above are just to get an initial idea of what the website will look like. I tried to stick with a very minimalistic colour palette in order to convey a sense of modesty. This will make the website more approachable since the UI is straightforward and doesn’t look over complicated.

The nav-bar will have a subtle grey-to black gradient. This will separate the nav-bar element from the background visually without creating a massive rift between the components. I wanted this partnership between the nav-bar and the main elements of the page to be subtle because the nav-bar determines what page is shown.

There will be elements that are colorful to convey the fact that they are interactable. I will make them responsive when the user hovers over them making it clear that an element is responsive to a user’s input.
