Initial Design




The idea of the design comes from a trend I came across with big gaming company websites (bellow will be links to each of these companies). My idea is that if this trend appears in all these big name companies, I can leverage off this trend to let this portfolio's intended auduence to know that I am intersted in thier field.
What I found very appealing in this design is the simlicity that it holds. Each deisgn element that is incorperated carries the message that the company wants to get through. Interaction with the website speaks to the user in a way that foreshadoes the users experience with thier product, which I find awesome.
